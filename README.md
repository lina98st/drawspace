# Drawspace

A freehand drawing app built with the HTML Canvas API.

Personal learning project focused on the Canvas 2D context, pointer events,
and state management in React.

## Features

- [ ] Freehand drawing
- [ ] Color picker
- [ ] Brush size
- [ ] Eraser
- [ ] Undo / Redo
- [ ] Clear canvas
- [ ] Export as PNG

## Tech Stack

- React
- TypeScript
- Vite
- Framer Motion
- CSS

## Getting Started

```bash
npm install
npm run dev
```
